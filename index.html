Meet GuardRag
Ask Questions Over Private Files, Locally.

GuardRag is a desktop Retrieval-Augmented Generation application that ingests your local documents, retrieves grounded context, and generates source-backed answers using local models.

Build Instructions
What GuardRag Does
  • Loads .pdf, .docx, .txt, .csv, and .json files
  • Builds semantic embeddings for retrieval
  • Runs a local LLM through Ollama
  • Shows answer sources and optional evaluation
  • Works as a local-first desktop workflow

About GuardRag

GuardRag is designed for private, practical question answering over your own files. It combines document indexing, semantic retrieval, and local generation so responses stay grounded in the uploaded content. The application includes a straightforward GUI, log tracking, and optional quality checks via evaluation metrics.
