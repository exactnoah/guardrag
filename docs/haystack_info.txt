Haystack is an open-source framework for building production-ready LLM applications, retrieval-augmented generative pipelines and state-of-the-art search systems that work intelligently over large document collections.

Key features of Haystack include:
- Pipeline-based architecture for composing NLP components
- Support for multiple document stores (Elasticsearch, FAISS, Weaviate, etc.)
- Integration with various LLM providers (OpenAI, Anthropic, Cohere, HuggingFace, etc.)
- Built-in components for retrieval, generation, and evaluation
- Easy to customize and extend

Haystack is maintained by deepset and has an active open-source community.
